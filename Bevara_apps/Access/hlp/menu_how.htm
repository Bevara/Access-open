How It Works

The Bevara Access software lets you playback or work with the data packaged in a bvr file. Each bvr file consists of data, such as a video clip, image, or document, packaged with an Accessor and optional metadata. The Accessor contains all of the information required to access or playback the associated data.
Scenario 1: Displaying a Preserved Image File
You have a bvr file containing a GIF image. The Bevara Access software reads the platform-independent Accessor and creates a player on-the-fly to display the image. If metadata is included in the bvr file, it can also be displayed.


Scenario 2: Extracting the Original Image Data and Metadata
You have a bvr file containing a GIF image. The bvr file format is open source so that the GIF image can easily be extracted in its original format to be used with any software that can display GIF images. Your metadata are also easily extracted for viewing in external applications.
