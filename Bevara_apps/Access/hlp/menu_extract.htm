Extract the Data or Metadata

A bvr file contains data and, optionally, metadata packed with an Accessor. These data can be extracted from the bvr file for use in other applications. Data extraction is a two-step process:
Select a bvr file by using the File Open icon.
Left-click on the Extract icon. You can set the filename and location for the extracted data in the pop-up dialog box.
Similarly, to extract the metadata, open the metadata for viewing using the two-step process:
Select a bvr file by using the File Open icon.
Left-click on the Info icon. The metadata will appear in a pop-up. Press the save button. You can enter the filename for the extracted metadata in the popup dialog.
